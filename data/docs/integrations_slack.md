# Slack Integration Guide

## Requirements

- Workspace Admin access
- Slack Workspace Administrator permissions

## Installation

1. Navigate to Settings > Integrations.
2. Select Slack.
3. Click Connect.
4. Authorize requested permissions.

## Common Errors

### Authorization Failed

Verify administrator permissions.

### Missing Notifications

Reconnect the integration and verify channel subscriptions.

### Delayed Messages

Slack API rate limiting may delay message delivery.
