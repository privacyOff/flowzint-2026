# Internal Support Playbook

## Escalate To Billing

When customers mention:

- refund
- invoice
- payment failed
- charged twice

## Escalate To Security

When customers mention:

- hacked
- compromised
- suspicious login
- data breach

## Escalate To Engineering

When customers report:

- API outage
- integration failures
- recurring platform bugs

## Escalate To Customer Success

When customers ask:

- onboarding assistance
- migration support
- training
