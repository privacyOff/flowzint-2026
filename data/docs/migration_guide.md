# Migration Guide

## Importing Users

Users may be imported using CSV files.

## Importing Historical Data

Supported formats:

- CSV
- JSON

## Recommended Process

1. Export data from existing platform.
2. Validate import format.
3. Run migration wizard.
4. Verify imported records.

## Common Issues

### Duplicate Users

Ensure email addresses are unique.

### Missing Records

Check file formatting and required fields.
