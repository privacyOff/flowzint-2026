# Getting Started

## Creating an Account

Users can sign up using email/password or SSO providers.

## Email Verification

New accounts must verify their email before accessing premium features.

## Workspace Creation

After registration, users are prompted to create their first workspace.

## Inviting Team Members

Workspace administrators can invite members from the Team Settings page.

## User Roles

- Owner
- Admin
- Agent
- Viewer

Each role has different permission levels.
