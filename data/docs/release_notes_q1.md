# Release Notes - Q1 2025

## January

### Workspace Templates
Customers can now create reusable workspace templates for onboarding new teams.

### Notification Center
A centralized notification center has been added to the dashboard.

## February

### Bulk User Import
Administrators can upload CSV files to create users in bulk.

### Audit Log Improvements
Audit logs now include role changes and login history.

## March

### Performance Improvements
Average dashboard load times were reduced by 35%.

### Security Updates
Improved MFA enrollment experience and security monitoring.
